# WebApplicationCRUD

Burn Book is a simple full-stack C# MVC web application that allows users to create, manage, and organize anonymous "tea" or fun school-related entries. The system follows CRUD functionality (Create, Read, Update, Delete) and demonstrates the integration of frontend, backend, and database components using ASP.NET MVC and SQL Server.

This project was developed as a school activity to demonstrate full-stack development concepts.
